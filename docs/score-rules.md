# Score Rules

하루 점수는 다이어트 목적의 생활 기록을 기준으로 산정한다.

- 총점: 100점
- 점수 대상: 운동, 식사, 음주, 마음, 독서
- 제외 대상: 무게
- 기본 배점: 항목별 20점 만점
- 가중치는 추후 조정 가능

## 확정된 기준

### 운동: 20점 만점

20점 조건:

- 강도 `보통`/`높음`: 운동시간 30분 이상
- 강도 `낮음`: 운동시간 40분 이상
- 운동 종류 입력

점수:

| 강도 | 조건 | 점수 |
| --- | --- | ---: |
| 보통/높음 | 30분 이상 | 20 |
| 보통/높음 | 20~29분 | 15 |
| 보통/높음 | 10~19분 | 10 |
| 보통/높음 | 1~9분 | 5 |
| 낮음 | 40분 이상 | 20 |
| 낮음 | 30~39분 | 15 |
| 낮음 | 20~29분 | 10 |
| 낮음 | 1~19분 | 5 |
| 전체 | 기록 없음 | 0 |

메모:

- 강도 `높음`은 추가 보너스가 아니라 20점 조건 충족으로 본다.
- 강도 미입력은 `보통/높음` 만점 조건을 충족하지 못하므로 30분 이상이어도 15점으로 본다.
- 운동 종류가 없으면 만점 조건을 충족하지 못하므로 최대 15점으로 본다.

### 식사: 20점 만점

다이어트 목적이므로 `점심 적정 + 저녁 소식`을 가장 좋은 패턴으로 본다.
두 끼 모두 소식은 단기적으로는 좋아 보일 수 있지만, 지속 가능성과 폭식 위험을 고려해 최고점보다 낮게 둔다.

| 점심 | 저녁 | 점수 |
| --- | --- | ---: |
| 적정 | 소식 | 20 |
| 소식 | 적정 | 18 |
| 적정 | 적정 | 17 |
| 소식 | 소식 | 14 |
| 적정 | 과식 | 10 |
| 과식 | 소식 | 9 |
| 소식 | 과식 | 8 |
| 과식 | 적정 | 7 |
| 과식 | 과식 | 3 |
| 기록 없음 | 기록 없음 | 0 |

메모:

- 저녁 과식을 점심 과식보다 더 크게 감점한다.
- `소식`은 나쁜 값이 아니라 다이어트에 유리한 선택으로 본다.
- 다만 두 끼 모두 소식은 지속 가능성 관점에서 만점으로 보지 않는다.

### 마음: 20점 만점

마음과 업무 컨디션을 각각 10점 만점으로 계산한다.
`좋음`과 `매우 좋음`은 같은 만점으로 본다.

| 조건 | 점수 |
| --- | ---: |
| 매우 나쁨 | 2 |
| 나쁨 | 5 |
| 보통 | 7 |
| 좋음 | 10 |
| 매우 좋음 | 10 |

예시:

| 마음 | 업무 컨디션 | 점수 |
| --- | --- | ---: |
| 좋음 | 매우 좋음 | 20 |
| 좋음 | 좋음 | 20 |
| 보통 | 좋음 | 17 |
| 보통 | 보통 | 14 |
| 나쁨 | 보통 | 12 |
| 매우 나쁨 | 나쁨 | 7 |

### 독서: 20점 만점

30페이지 이상 읽으면 만점으로 계산한다.

| 조건 | 점수 |
| --- | ---: |
| 30페이지 이상 | 20 |
| 20~29페이지 | 15 |
| 10~19페이지 | 10 |
| 1~9페이지 | 5 |
| 기록 없음 | 0 |

메모:

- 독서 시간은 참고 정보로 남기고, 점수는 페이지 기준으로 계산한다.
